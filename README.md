# Shipping Partner Portal (UI scaffold)

This folder contains a Vite + React + TypeScript scaffold with Tailwind CSS prepared for the Shipping Partner Portal login and onboarding flow.

To run locally:

1. cd shipping-portal
2. npm install
3. npm run dev

Notes:
- The project currently simulates email verification (enter code `123456`).
- Implement real authentication and backend API integration as needed.
- Tailwind is configured in tailwind.config.cjs and index.css includes the Tailwind directives.
